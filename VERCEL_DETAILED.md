# Подробная инструкция: Деплой на Vercel

## Часть 1: Регистрация и подключение GitHub

### 1.1 Откройте Vercel

1. Перейдите на https://vercel.com
2. Нажмите кнопку **"Sign Up"** (в правом верхнем углу)

### 1.2 Авторизация через GitHub

1. На странице регистрации выберите **"Continue with GitHub"**
2. Если вы не залогинены в GitHub - введите логин/пароль
3. GitHub попросит разрешение для Vercel - нажмите **"Authorize Vercel"**
4. Vercel может попросить подтвердить email - проверьте почту и подтвердите

✅ Теперь вы в панели Vercel

---

## Часть 2: Импорт проекта

### 2.1 Создание нового проекта

1. На главной странице Vercel нажмите **"Add New..."** (справа сверху)
2. В выпадающем меню выберите **"Project"**

### 2.2 Импорт репозитория

1. Вы увидите список ваших GitHub репозиториев
2. Найдите **"MTProxy_Manager"** в списке
   - Если не видите - нажмите **"Adjust GitHub App Permissions"** и дайте доступ к репозиторию
3. Напротив **"MTProxy_Manager"** нажмите кнопку **"Import"**

### 2.3 Настройка проекта

Откроется страница "Configure Project". Здесь:

1. **Project Name**: оставьте `mtproxy-manager` (или измените на свое)
2. **Framework Preset**: должно автоматически определиться как **"Next.js"**
3. **Root Directory**: оставьте `./` (корень проекта)
4. **Build and Output Settings**: оставьте по умолчанию

---

## Часть 3: Настройка переменных окружения

### 3.1 Откройте раздел Environment Variables

1. Прокрутите страницу вниз до раздела **"Environment Variables"**
2. Вы увидите поля для добавления переменных

### 3.2 Добавьте первую переменную: API_SECRET

1. В поле **"Key"** (слева) введите:
   ```
   API_SECRET
   ```

2. В поле **"Value"** (справа) введите:
   ```
   AcKx5ZzCotRghOYnl7p1W2DwEFyPrU8T
   ```

3. Оставьте выбранными все окружения: **Production**, **Preview**, **Development**

4. Нажмите **"Add"** (или просто Enter)

### 3.3 Добавьте вторую переменную: ADMIN_PASSWORD

1. В поле **"Key"** введите:
   ```
   ADMIN_PASSWORD
   ```

2. В поле **"Value"** введите ваш пароль (придумайте надежный):
   ```
   MySecurePassword2026!
   ```
   ⚠️ Запомните этот пароль - он нужен для входа в панель!

3. Нажмите **"Add"**

### 3.4 Добавьте третью переменную: DATABASE_PATH

1. В поле **"Key"** введите:
   ```
   DATABASE_PATH
   ```

2. В поле **"Value"** введите:
   ```
   ./data/proxy.db
   ```

3. Нажмите **"Add"**

### 3.5 Проверьте переменные

Должно быть 3 переменные:
- ✅ API_SECRET = AcKx5ZzCotRghOYnl7p1W2DwEFyPrU8T
- ✅ ADMIN_PASSWORD = (ваш пароль)
- ✅ DATABASE_PATH = ./data/proxy.db

---

## Часть 4: Деплой

### 4.1 Запуск деплоя

1. Прокрутите страницу вниз
2. Нажмите большую синюю кнопку **"Deploy"**

### 4.2 Ожидание

1. Начнется процесс сборки (Building)
2. Вы увидите логи в реальном времени
3. Процесс займет 2-3 минуты

Этапы:
- ⏳ Cloning repository
- ⏳ Installing dependencies
- ⏳ Building application
- ⏳ Deploying

### 4.3 Успешный деплой

Когда деплой завершится, вы увидите:
- 🎉 Конфетти на экране
- ✅ "Congratulations!" или "Your project has been deployed"
- Превью вашего сайта

---

## Часть 5: Получение URL

### 5.1 Скопируйте URL проекта

1. На странице успешного деплоя найдите URL вашего проекта
   - Обычно это что-то вроде: `https://mtproxy-manager-abc123.vercel.app`
2. Нажмите на URL или кнопку **"Visit"** чтобы открыть панель
3. **СКОПИРУЙТЕ ЭТОТ URL** - он понадобится для настройки сервера!

### 5.2 Сохраните данные

Запишите в безопасное место:

```
URL панели: https://mtproxy-manager-________.vercel.app
API_SECRET: AcKx5ZzCotRghOYnl7p1W2DwEFyPrU8T
ADMIN_PASSWORD: ваш_пароль
```

---

## Часть 6: Проверка работы панели

### 6.1 Откройте панель

1. Перейдите по URL вашего проекта
2. Вы должны увидеть интерфейс **"MTProxy Manager"**
3. Пока прокси нет - таблица будет пустая

### 6.2 Что должно быть видно

- ✅ Заголовок "MTProxy Manager"
- ✅ Кнопки "Ротация всех" и "Добавить прокси"
- ✅ Карточки статистики (все нули)
- ✅ Пустая таблица прокси
- ✅ Панель логов (пустая)

Если видите ошибку:
- Проверьте логи в Vercel Dashboard → Functions → Logs
- Убедитесь, что все 3 переменные окружения добавлены

---

## Часть 7: Настройка домена (опционально)

Если хотите свой домен вместо `*.vercel.app`:

1. В Vercel Dashboard откройте ваш проект
2. Перейдите в **Settings → Domains**
3. Добавьте свой домен
4. Настройте DNS записи (Vercel покажет инструкции)
